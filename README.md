# 🚀 Task Manager Full Stack

Este projeto é uma aplicação de lista de tarefas completa, integrando um backend em **ASP.NET Core (.NET 10)** com um frontend moderno em **Vue.js 3**.

## 🛠️ Tecnologias Utilizadas
- **Backend:** .NET 10 Web API (C#)
- **Frontend:** Vue.js 3 + Vite + Axios
- **Versionamento:** Git & GitHub
